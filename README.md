# To Do List

## Steps to follow : <br><br>

### If the user is new , start from <b><u>Step 1</u> </b> else the user can start from <b><u>Step 2</u> </b> </br> </br>

### <b><u>Step 1</u> </b> : In the <b>Log In</b> page click on " Sign Up " button to get to Sign Up page . There the user must fill in all the input fields and press either " Enter " key or "Register" button or else failure message will show up on the top of the page . After successful sign up the user will be redirected to the login page again . The user can also press the " Go Back " button to get back to the " Log In " page .</br></br>

### <b><u>Step 2</u></b> : In the login page fill up both input boxes and press the " Log In " button. If the login is unsuccessful then an error message will occur at the top. Else the user will be redirected to the <b>Insert</b> page</br></br>

### <b><u>Step 3</u></b> : In the <b> Insert </b> page we have two buttons " Enter " and " Log out " . If the user wants to get to the <b>Log In</b> page press the " Log Out " button . Else to enter the list title put list title on the input field and press " Enter " key , If the title has been created before then the title won't be recreated , so create titles with unique names . All the titles that you have created will be displayed in "All titles" section below the input field.</br></br>

### <b><u>Step 4</u></b> : Click on the list title to view the <b>List</b> page where you can enter items to-do . </br> Enter items into the input box and press "Enter" or "+" to add items and to delete items press on the checkbox . You can also press " Go Back " button to return to the <b> Insert </b> page .

</br></br>
